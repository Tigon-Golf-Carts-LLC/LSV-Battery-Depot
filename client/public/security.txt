## RESPONSIBLE DISCLOSURE POLICY
LSV Battery Depot is committed to the security of our website and customer data. We welcome responsible security research and encourage the responsible disclosure of security vulnerabilities.

### Scope of Security Testing
Authorized Testing Areas:
✓ Public website content and functionality (lsvbatterydepot.com)
✓ Customer-facing forms and input fields
✓ Public API endpoints and responses
✓ Web application security scanning
✓ SSL/TLS configuration analysis

### Prohibited Testing Activities
✗ Testing that disrupts service availability
✗ Social engineering attacks on employees
✗ Physical security testing of facilities
✗ Denial of service (DoS) attacks
✗ Automated vulnerability scanning without permission
✗ Testing of third-party integrated services
✗ Data destruction or modification attempts

### Vulnerability Reporting Process
1. Identify and document the security issue
2. Contact our security team at 1-844-888-7732
3. Provide detailed vulnerability information
4. Allow reasonable time for investigation and remediation
5. Coordinate public disclosure timing if applicable

### Required Vulnerability Information
Please include the following in your security report:
- Detailed description of the vulnerability
- Steps to reproduce the issue
- Potential impact and risk assessment
- Proof of concept (if applicable)
- Suggested remediation approaches
- Your contact information for follow-up

## LEGAL CONSIDERATIONS

### Authorized Security Testing
Security researchers are authorized to test our public website within the scope defined above. Any testing outside this scope requires explicit written permission.

### Safe Harbor
LSV Battery Depot will not pursue legal action against security researchers who:
- Follow our responsible disclosure policy
- Make good faith efforts to avoid data destruction
- Do not access or modify customer data
- Report vulnerabilities through proper channels
- Allow reasonable time for remediation
